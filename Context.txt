Architecture Document: Threshold-Triggered Route Rotation
Alternative approach focusing on extreme computational efficiency and explicit logic.
1. The Core Concept: The "Three Strikes" Routing Stack
Instead of running expensive microsimulations to predict cascading failures, we rely on historical hierarchy and strict rule-based penalization to handle both Unplanned (accidents, breakdowns) and Planned (rallies, sports matches) events within a unified engine.
Every corridor/zone intersection has a predefined Route Stack (a list of diversion plans ranked by historical efficiency).
If the Primary route fails a set number of times (the threshold), it receives a penalty, is temporarily demoted, and the Secondary route becomes the new default.
Primary: Route B-D-E-A (Default)
Secondary: Route B-D-E-C (Fallback 1)
Tertiary: Route B-D-E-P (Fallback 2)

5. Resolving Pain Point 3: The Evolutionary Learning Loop
A. The Failure Metric (The Strike)
A route receives "1 Strike" if the live execution results in , OR if a secondary incident is logged on the active diversion route.
B. The Threshold Trigger & Rotation
If a route hits the 3-Strike Threshold within a rolling 14-day window:
Demotion: The route is moved to the "Penalty Box" (demoted to Tertiary).
Promotion: The Secondary route is promoted to Primary.
Resource Update: The system recalculates the Resource Matrix for the new Primary route.
C. The Rehabilitation Protocol (Safe Re-entry)
After a 30-day cooldown, a demoted route enters Rehabilitation Mode.
The system will automatically test the penalized route only if a matching event occurs during historically low-volume hours (e.g., 11:00 PM - 4:00 AM).
If it succeeds without strikes, it slowly regains its rank.
6. Contextual Sub-Stacks (Preventing Cross-Contamination)
To prevent a massive planned event from penalizing a route used for daily commuting emergencies, stacks are strictly siloed based on context and scale:
Stack Alpha (Unplanned / Clear Weather / Off-Peak)
Stack Beta (Unplanned / Rain / Peak Hours)
Stack Gamma (Planned / High-Volume > 10k attendees)
Strikes are strictly isolated to the active context stack. If a diversion route fails during a 50,000-person rally, it receives a strike in Stack Gamma, but remains the trusted Primary route in Stack Alpha for a random Tuesday fender-bender.
